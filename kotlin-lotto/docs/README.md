# 기능 명세서

객체와 책임, 그리고 역할을 작성.

## Lottos - 로또의 모음집 역할

- [ ] 당첨 로또인 지 판별한다.

## Lotto - 6개의 숫자의 모음집 역할

- [ ] 6개의 숫자가 중복되지 않는 숫자들로 이루어져 있는지 확인한다.
- [ ] 로또 숫자를 보여줄 때 로또 번호를 오름차순으로 정렬한다.

## Number - 숫자가 해야 할 역할

- [ ] 숫자의 범위가 1 ~ 45인지 확인한다.

## View

- [ ] 유저의 입력을 받는다.
- [ ] 유저에게 출력한다.

## Controller

- [ ] View와 Model 사이의 연결을 한다.
