# 기능 요구사항

## 로또 게임 Rule 요구사항
- [x] 로또 금액 입력 받기
  - 구입 금액은 1,000원 단위
  - **(예외)** 1,000원으로 나누어 떨어지지 않는 경우 예외 처리 
- [ ] 당첨 번호 및 보너스 번호 입력 받기 
  - 번호는 쉽표를 기준으로 구분
  - **(예외)** 번호가 1미만 ~ 45초과일 경우 예외 처리
- [x] 금액 만큼 로또를 생성
- [x] 로도 수량 및 번호 출력
  - 로또 번호 오름차순 정렬
- [ ] 당첨 통계 계산
- [ ] 당첨 내역 출력
- [ ] 수익률 계산
- [ ] 수익률 출력
  - 수익률 소숫점 둘재 자리에서 반올림

## 예외 요구 사항
- [ ] 예외 상황 발생시 에러 출력
  - 에러 문구는 "[ERROR]"로 시작

## 코틀린 컨벤션
- setter를 열어두되 사용하지 않는다. var로 프로퍼티를 선언하는 경우, 별도의 함수를 통해서 프로퍼티 값을 변경한다.
